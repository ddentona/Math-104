\section{Problem 1}
Prove that convergence of $\{s_n\}$ implies convergence of $\{\abs{s_n}\}$. Is the converse true?
(Baby Rudin, Problem 3.1, p.78)

\pagebreak
\section{Problem 2}
Calculate $\lim_{n \to \infty} \left( \sqrt{n^2 + n} - n \right)$.
(Baby Rudin, Problem 3.2, p.78)

\pagebreak
\section{Problem 3}
If $s_1 = \sqrt{2}$, and $s_{n + 1} = \sqrt{s + \sqrt{s}}$ (n = 1, 2, 3, ... ), prove that ${s_n}$ converges, and that $s_n < 2$ for n = 1, 2, 3, ...
(Baby Rudin, Problem 3.3, p.78)

\pagebreak
\section{Problem 4}
Find the upper and lower limits of the sequence $\{s_n\}$ defined by $s_1 = 0; s_{2m} = \frac{s_{2m - 1}}{2}; s_{2m + 1} = \frac{1}{2} + s_{2m}$.
(Baby Rudin, Problem 3.4, p.78)

\pagebreak
\section{Problem 5}
For any two real sequences $\{a_n\}$, $\{b_n\}$, prove that
\begin{equation}
    \limsup_{n \to \infty} (a_n + b_n) \leq \limsup_{n \to \infty} a_n + \limsup_{n \to \infty} b_n,
\end{equation}
provided the sum on the right is not of the form $\infty - \infty$.
(Baby Rudin, Problem 3.5, p.78)

\pagebreak
\section{Problem 6}
Investigate the behavior (convergence or divergence) of $\sum a_n$ if
\begin{enumerate}[label=(\alph*)]
    \item   $a_n = \sqrt{n + 1} - \sqrt{n}$
    \item   $a_n = \frac{\sqrt{n + 1} - \sqrt{n}}{n}$
    \item   $(\sqrt[n]{n} - 1)^n$
    \item   $a_n = \frac{1}{1 + z^n}$ for complex $z$.
\end{enumerate}
(Baby Rudin, Problem 3.6, p.78)
